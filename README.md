# 飞机大战 (Aircraft War)

基于 Java Swing 的经典飞机射击游戏，哈尔滨工业大学（深圳）面向对象程序设计课程项目。

## 游戏特性

- **三种难度**：简单 / 普通 / 困难模式，普通及以上难度随游戏时间动态提升敌机强度
- **5 种敌机**：普通敌机、精英机、精英 Plus、精英 Pro、Boss 机
- **5 种道具**：回血、炸弹（全屏清敌）、火力增强、火力 Plus、冰冻（暂停敌机移动）
- **分数排行榜**：基于文件持久化存储，支持增删查
- **音效与背景音乐**：普通/Boss 两套 BGM，击中、爆炸、游戏结束等多种音效

## 涉及的面向对象设计模式

| 模式     | 应用位置                               |
|--------|------------------------------------|
| 模板方法   | `AbstractGame` 定义游戏主循环骨架           |
| 工厂方法   | `EnemyAircraftFactory` 及其子类创建各类敌机  |
| 简单工厂   | `PropSimpleFactory` 随机生成道具         |
| 策略模式   | `ShootStrategy` 定义多种射击方式（直射、扇形、环形） |
| 单例模式   | `HeroAircraft` 唯一玩家飞机实例            |
| 观察者模式  | `PropSubject` 炸弹道具通知所有敌机销毁         |
| DAO 模式 | `ScoreDaoByFile` 文件读写分数记录          |

## 项目结构

- `src/` — 源代码、图片资源、音视频资源
- `test/` — JUnit 5 单元测试
- `lib/` — 第三方依赖
- `uml/` — PlantUML 类图与设计文档
